# Summary

A simple button to make it easier to change filament by pressing the screw on the extruder to release the filament and pull it out/put it in. Your fingers will like it. ;-)

Should be printed with 100% infill!



# Print Settings

Printer: Anet A8
Rafts: No
Supports: No
Resolution: 0.2mm
Infill: 100%
