# Summary

A cable clip designed for 2040 extruded aluminum bars.  The OpenScad file is fully parametric.  Adjust any of the following to customize the clip:
loop_diameter = 17; for sizing of the clip top
loop_height = 15; for adjusting the overall length of the clip
hook_length = 5; for adjusting the hook size
hook_thickness = 1; for adjusting the wall and hook thickness.
lip_size = 1;  for the side length of a rectangular prism on the end of the hook.
part_height = 10; for the overall height of the piece
quality = 100; for the quality of the cylinders in the design
